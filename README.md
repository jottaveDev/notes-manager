# notes-manager
Sistema de gerenciamento de notas escolares. A solução perfeita para professores gerenciar suas turmas e alunos.

### Funcionalidades
 - Cadastro de turmas
 - Cadastro de alunos
 - Registrar, consultar, editar, e excluir notas/alunos/turmas

### Tecnologias utilizadas
 - Java 17
 - SQL
